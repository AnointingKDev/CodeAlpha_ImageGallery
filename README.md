CodeAlpha Intership Task 01 — Image Gallery

Overview

A fully responsive Image Gallery built with HTML, CSS, and JavaScript. The application allows users to browse images in an elegant gallery layout, filter images by category, search images instantly, and view them in a lightbox with navigation controls.

Features

* Responsive gallery layout
* Category-based image filtering
* Live image search
* Lightbox image preview
* Previous and Next navigation
* Keyboard navigation support
* Mobile-friendly touch interactions
* Hover effects and smooth animations
* Responsive design across all devices

Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

Project Structure

```text
CodeAlpha_ImageGallery/
│
├── index.html
├── style.css
├── script.js
└── images/
```

Getting Started

1. Clone the repository

```bash
git clone https://github.com/YourUsername/CodeAlpha_ImageGallery.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.

No installation or dependencies are required.
